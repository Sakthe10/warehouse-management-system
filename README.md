# Warehouse Management System

## Overview
A MySQL-based Warehouse Management System designed to manage products, inventory, warehouses, customers, suppliers, purchase orders, sales orders, deliveries, and payments.

## Technologies Used
- MySQL
- SQL
- MySQL Workbench
- GitHub

## Features
- Product Management
- Inventory Tracking
- Warehouse Management
- Customer Management
- Supplier Management
- Purchase Orders
- Sales Orders
- Delivery Tracking
- Payment Management

## Database Concepts Used
- Primary Keys
- Foreign Keys
- Joins
- Aggregate Functions
- Group By
- Having
- Subqueries
- Window Functions
- Views
- Triggers

## How to Run
1. Open MySQL Workbench.
2. Import the `warehouse_management.sql` file.
3. Execute the script to create the database and tables.
4. Run SQL queries for analysis and reporting.
